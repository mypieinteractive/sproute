Sproute FAQ

Everything you need to know about our inspection-first logic.

What is Sproute?

Sproute is an order management and route planning software, built BY inspectors, FOR inspectors. There are lots of similar products on the market, but Sproute is different in that it is uniquely built around the needs of regional property inspection companies—not delivery drivers or huge fleet operators.

This approach means we can offer advanced features specific to the needs of the property inspection community, while keeping things streamlined so that the monthly cost remains well below other products of similar quality.

"Go ahead and compare. We dare you to find a cheaper service."

What is an API call, and why should I care?

Sproute does not perform address validation or route optimization directly. Instead, we tap into the "gold-standard" resources of Google Maps and Mapbox through an API (Application Programming Interface).

Because these providers charge a fee for every call, Sproute uses a clever tiered approach and custom backend coding to keep your costs effective while utilizing the strengths of both platforms.

1. Address Geocoding & Validation

Sproute Address Cache $0 API COST

The first place Sproute looks. If the address is matched here, it costs you nothing.

Mapbox Geocoding API CALL PER ADDRESS

The simplest and most efficient call. Once matched, we save it to the cache so you never pay for that specific address again.

Google Address Validation API CALL PER ADDRESS

The "heavy hitter" used for difficult rural areas or addresses with typos.

Manual Validation

If automation fails, we give you the tools to sleuth and enter coordinates manually. Once saved, it’s cached forever.

2. Map Generation

Every time a map is generated or reordered to help you manage pins, a single API call is utilized to ensure your visual data is live and accurate.

3. Route Optimization

Solving the "Traveling Salesman" problem is the most complex task. We use a tiered approach here as well:

Google Routes (Up to 25 stops)

The most cost-effective method for standard daily routes.

Google Route Optimization (25+ stops)

The most powerful logic available, capable of handling hundreds of stops at once.

Re-optimization and Heuristic Math

We know routes change. To save your API limits, Sproute uses heuristic math (K-means clustering) when you make manual edits. This allows us to recalculate distances and times instantly without hitting a paid API.

Administrators have full control and can trigger a full "hard" re-optimization whenever necessary, or grant that power to inspectors in the field via the Settings menu.
